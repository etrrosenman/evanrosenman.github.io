Projects
  • ProseMapper (poems and articles)
    • ProseMapper is a project designed to visualize a corpus of documents in a user-friendly way. It used on this website to display both my poetry and my Crimson articles such that a viewer can explore my writing by getting a sense of a poem or article's topic prior to reading it. ProseMapper encompasses a series of programs which take in a set of text documents, calculate their pairwise similarities, calculate an optimal spatial layout for nodes representing each document, and visualize these nodes. LinLogLayout is used for node clustering and Arbor.js is used for node display. Information about how this project was designed and coded can be found here.
  • JobLink
    • JobLink--completed in a team with Olga Zinoveva and Matthew Chartier as a final project for Computer Science 179--is an mobile app designed to aid students in the recruiting process. The idea for the project emerged from our collective observation that students often seek information informally in order to discover potential employers and to learn about interviewing. We designed an algorithm to determine companies' similarity and visualize these connections in a user-friendly interface, and also used machine learning to generate company recomendations for students seeking employment. The site layout was generated using jQuery and jQuery mobile, while the graph was generated using arbor.js. Noes were laid out using an algorithm from LinLogLayout.
